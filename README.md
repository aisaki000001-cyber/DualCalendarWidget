# 双月日历 - macOS 通知中心小组件

一个轻量级 macOS 日历小组件：双月并排 + 农历 + 中国节假日调休标注。

## 功能

- 📅 **双月日历**：当月和下月并排显示，一目了然
- 🌙 **农历日期**：每个日期下方显示农历（初一、初二…）
- 🎋 **节日节气**：春节、端午、中秋、除夕、父亲节、廿四节气…
- 🇨🇳 **法定节假日**：2026 国务院官方放假安排，含连休和调休补班
- ⬅️➡️ **翻页浏览**：前后 12 个月随意翻
- 🔵 **今日标记**：蓝色圆圈高亮今天
- 🟢 **绿点**：放假休息日 | 🔴 **红点**：调休补班日
- 🖥️ **通知中心**：触控板右滑即可查看，无需打开 App

## 安装

1. 从 [Releases](https://github.com/aisaki000001-cyber/DualCalendarWidget/releases) 下载 `双月日历小组件.dmg`
2. 打开 DMG，把 `DualCalendarWidget` 拖到 `Applications` 文件夹
3. 运行一次 App（加载节假日数据）
4. 触控板右滑 → 通知中心 → 编辑小组件 → 搜索「双月日历」→ 添加

> **提示**：首次打开如提示「无法验证开发者」，右键点击 App → 打开 即可。
